# 🚀 5 分钟部署指南

## 第一步：推送到 GitHub (2 分钟)

```bash
cd /Users/lulu/.openclaw/workspace/ai-news-agent

# 初始化 git
git init
git add .
git commit -m "🦞 AI 资讯选题系统初始化"

# 关联远程仓库
git remote add origin https://github.com/lalahoney11-pixel/ai-news-workspace.git
git branch -M main
git push -u origin main
```

**如果提示仓库不存在：**
1. 打开 https://github.com/new
2. 仓库名：`ai-news-workspace`
3. 设为 **Public** (Actions 免费)
4. 创建后重复上面的 git 命令

---

## 第二步：配置 Secrets (2 分钟)

### 2.1 获取 GitHub Token

1. 访问 https://github.com/settings/tokens
2. 点击 **Generate new token (classic)**
3. 名称：`ai-news-actions`
4. 勾选 **repo** (全部子选项)
5. 生成并复制 Token (以 `ghp_` 开头)

### 2.2 获取 AI API Keys (可选但推荐)

**Google Gemini:**
1. 访问 https://aistudio.google.com/app/apikey
2. 点击 **Create API Key**
3. 复制 Key

**Groq:**
1. 访问 https://console.groq.com/keys
2. 点击 **Create API Key**
3. 复制 Key

### 2.3 添加到 GitHub Secrets

1. 打开你的仓库：https://github.com/lalahoney11-pixel/ai-news-workspace
2. 进入 **Settings → Secrets and variables → Actions**
3. 点击 **New repository secret**
4. 添加以下 Secrets：

| Name | Value |
|------|-------|
| `GITHUB_TOKEN` | 你刚才生成的 GitHub Token |
| `GITHUB_REPO` | `lalahoney11-pixel/ai-news-workspace` |
| `GEMINI_API_KEY` | (可选) Gemini API Key |
| `GROQ_API_KEY` | (可选) Groq API Key |

---

## 第三步：测试运行 (1 分钟)

1. 打开仓库的 **Actions** 标签
2. 点击左侧 **Daily AI News Fetch**
3. 点击 **Run workflow**
4. 等待 1-2 分钟

运行成功后：
- ✅ 你会看到绿色的成功标记
- ✅ 仓库的 **Issues** 里会出现新的选题

---

## 第四步：设置看板 (可选)

1. 在仓库顶部点击 **Projects** → **New project**
2. 选择 **Board** 模板
3. 添加列：
   - 📥 待处理
   - ✍️ 写作中
   - ✅ 已发布
4. 把 Issues 拖拽到对应列

## 自动运行

系统会每天 **北京时间 8:00** 自动执行：
- 抓取最新资讯
- AI 评估打分
- 发布 7 分以上内容到 Issues

你醒来就能看到当天的选题清单！

---

## 常见问题

**Q: Actions 没有运行？**
A: 检查 Settings → Actions → General，确保 Actions 已启用

**Q: 没有 API Key 能用吗？**
A: 可以！系统会降级使用关键词打分，但准确度会降低

**Q: 想修改运行时间？**
A: 编辑 `.github/workflows/daily-fetch.yml` 中的 cron 表达式

**Q: 想调整分数阈值？**
A: 在 Secrets 中添加变量 `MIN_SCORE` (默认 7)
